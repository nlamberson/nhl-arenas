# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v56.0.0/ before writing any code.

## Frontend layout

- `src/app/` — Expo Router screens and layouts (routing only)
- `src/components/` — app UI (VisitRow, NHLBrand, MatchupScoreboard, etc.)
- `src/components/ui/` — shadcn design system (Button, Input, Label, Text)
- `src/lib/`, `src/hooks/`, `src/context/` — domain logic and data fetching
- `assets/` — images and fonts (project root; referenced via `@/assets/*`)
